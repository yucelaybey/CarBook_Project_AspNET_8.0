* AspNet Core 8.0
* Asp.Net Core Api
* Asp.Net Identity
* Entity Framework Core
* MSSQL
* X.PagedList ile Pagination İşlemleri
* Onion Architecture
* Mediator Design Pattern
* Swagger
* Postman
* Api Consume
* Json Web Token
* Login
* Register
* DTO Layer
* Mvc
* Deploy
* Tüm CRUD İşlemleri
* SignalR (sadece bir yerde test edilmiştir.)
